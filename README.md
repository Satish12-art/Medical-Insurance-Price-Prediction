# 💰 Medical Insurance Cost Prediction

This project predicts medical insurance cost based on user details using Machine Learning.

## 🚀 Live Demo
👉 https://medical-insurance-price-prediction-emybwvnpupugti9vzle6mr.streamlit.app

---

## 📌 Project Overview
- Built an end-to-end Machine Learning project  
- Trained model using real-world insurance dataset  
- Deployed as a web app using Streamlit  

---

## 🧠 Features Used
- Age  
- BMI  
- Number of Children  
- Gender  
- Smoking Status  
- Region  

---

## ⚙️ Tech Stack
- Python  
- Pandas  
- NumPy  
- scikit-learn  
- Streamlit  

---

## 📊 Model Details
- Used Linear Regression & Decision Tree  
- Evaluated using RMSE  
- Achieved approx error of ~6000  

---

## 🔍 Key Insights
- Smoking has the highest impact on insurance cost  
- BMI and Age also significantly affect cost  
- Region and gender have minimal impact
